It’s a Tracker!

Boo! I’m a naughty Web page trying to spy on you!

OK, not really, but Web trackers are out there on the Internet. Firefox provides built-in Tracking Protection in Private Browsing to help you block Web trackers.

If you're using Firefox 42 or later with Tracking Protection enabled, you should see a shield icon in the URL bar and both of these should be green:

  • a third-party tracker was ...correctly blockedincorrectly loaded
  • a first-party tracker was ...correctly loadedincorrectly blocked
